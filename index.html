MindBehindIT MCP Proxy

Version: 0.3.2

Download Download MindBehindIT-MCP-Proxy.zip

The ZIP file is password-protected. Obtain the password from your administrator or release notes.

Installation
  1. Download the ZIP file above
  2. Unzip using the password provided by your admin
  3. Open the DMG and drag the app to Applications
  4. In Terminal, run: xattr -cr "/Applications/MindBehindIT MCP Proxy.app"
  5. Launch the app from Applications
