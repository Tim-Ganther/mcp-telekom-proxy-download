// MCP Proxy

MindBehindIT MCP Proxy

Version 0.5.1

The MCP Proxy connects Open WebUI and other MCP clients with your internal services – secure, local, and reliable.

One-line installer

Open Terminal and run this command. You will be prompted for the ZIP password.

curl -fsSL https://Tim-Ganther.github.io/mcp-telekom-proxy-download/install.sh | bash

Password: The ZIP is password-protected. Obtain the password from your administrator or the release notes.

Manual download

Download MindBehindIT-MCP-Proxy.zip — unzip with the password, open the DMG, and drag the app to Applications. Then run xattr -cr "/Applications/MindBehindIT MCP Proxy.app" in Terminal.
